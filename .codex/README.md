这个项目包含供 Codex 使用的 AI Skills。
Agent Skills for Ben's AI in VS Code (Codex).
These skills follow the [Agent Skills specification](https://agentskills.io/specification).

## Installation
复制 `skills/` 到 Codex 的 skills 路径（通常为 `~/.codex/skills`）。
